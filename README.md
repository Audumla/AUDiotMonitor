# AUDiotMonitor

A cross-platform desktop audio monitoring application built with Electron and React using TypeScript.

## Features
- Real-time audio monitoring
- Modern UI with React
- Cross-platform support (Windows, macOS, Linux)

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm or yarn

### Installation
```sh
npm install
```

### Development
```sh
npm run dev
```

### Build
```sh
npm run build
```

## Project Structure
- `src/` - Source code for main and renderer processes
- `public/` - Static assets

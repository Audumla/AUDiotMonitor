# Adapters Reading Path

Use this path when implementing or updating adapters.

## Required

1. [System Architecture](../foundation/spec-100-component-architecture.md)
2. [Spec 101 (storage/system)](../adapters/spec-101-deep-hardware-telemetry.md)
3. [Spec 801 (gateway manifest)](../adapters/spec-801-llmgateway-monitor-adapter.md)

## Quick Example

For a new adapter source type:
1. Define adapter behavior in adapter spec.
2. Add interface/lifecycle rules in `exporter-interface-spec.md`.
3. Add implementation checklist entry in `implementation-plan-stages.md`.
